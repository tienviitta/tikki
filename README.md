# Tikki

A card game built with vanilla JavaScript using the [Deck of Cards API](https://deckofcardsapi.com/).

## Features

- Fetch and shuffle a new deck from Deck of Cards API
- Deal 10 cards alternating between player and computer
- Interactive card game interface

## Getting Started

### Prerequisites

- Node.js (for development server)

### Installation

1. Clone the repository

```bash
git clone https://github.com/tienviitta/tikki.git
cd tikki
```

2. Install dependencies

```bash
npm install
```

3. Start the development server

```bash
npm run dev
```

4. Open your browser and navigate to the local server URL

## Usage

Click the "Deal" button to shuffle a new deck and deal cards to both player and computer hands.

## Tech Stack

- Vanilla JavaScript
- [Vite](https://vite.dev/) - Build tool
- [Deck of Cards API](https://deckofcardsapi.com/) - Card deck management
